Pour tester l'app :
* Cloner le repo
* `bundle install`
* `rails db:migrate`
* `rails db:seed`
* `rails server`
* Se rendre sur localhost:3000/ dans le navigateur
* Enjoy ce super site

Ou bien se rendre sur https://mutmutbrite.herokuapp.com/ une fois que le front sera mis en place :)
